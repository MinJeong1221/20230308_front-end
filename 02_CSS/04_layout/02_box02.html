CSS 여백과 테두리

Box 모양으로 margin(바깥 여백), border(테두리), pading(안쪽 여백), content(컨텐츠 영역) 이렇게 있다.

1. border(테두리)

border은 박스의 테두리의 두께, 모양, 색상 지정할 수 있다.(단축속성)
border를 지정하면 요소의 너비에 영향을 미친다.

1-1 border 개별속성

  • border-width : 선의 두께
  • border-style : 선의 모양
  • border-color : 선의 색상
  • border-right : 오른쪽
  • border-left : 왼쪽
  • border-radius : 모서리를 둥글게 처리
    -- border-top-right-radius : 각 모서리 별로 처리도 가능

2. 바깥 여백 margin

요소의 바깥 여백을 지정한다(시계방향)

margin이 겹치는 부분이 합쳐지는 경우가 있다(마진상쇄)

margin을 지정해도 너비에 영향을 미치지 않는다

너비가 지정되어있고 좌우너비를 auto로 두면 가운데로 보낼수 있다. (margin: 0 auto;)

2-1. margin 단축속성

margin : 10px(상하좌우 다 적용)된다.

margin : 10px 20px(상하 좌우)이렇게 적용된다.

margin : 10px 20px 30px(상 좌우 하)이렇게 적용된다.

margin : 10px 20px 30px 40px(상 우 하 좌)이렇게 적용된다.

2-2. margin 개별속성

margin-top, margin-right, margin-left, margin-bottom ...등

3. 안쪽 여백 paddig

요소의 안쪽 여백을 지정한다(시계방향)

padding을 지정하면 너비에 영향을 미친다

3-1. padding 단축속성

paddig : 10px(상하좌우 다 적용)된다.

paddig : 10px 20px(상하 좌우)이렇게 적용된다.

paddig : 10px 20px 30px(상 좌우 하)이렇게 적용된다.

paddig : 10px 20px 30px 40px(상 우 하 좌)이렇게 적용된다.

3-2. padding 개별속성

padding-top, padding-right, padding-left, padding-bottom ...등

4. box-sizing

요소의 너비와 높이계산 방법을 지원한다

4-1. box-sizing : content-box(기본속성)

너비와 높이가 content영역이 기준이다

4-2. box-sizing : border-box

너비와 높이가 border까지의 영역이 기준이 된다
