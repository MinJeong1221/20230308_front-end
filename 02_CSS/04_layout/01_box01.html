CSS Box

1. 블록 레벨 요소(Block Level Element)

- 블록레벨 요소는 영역을 지정할때 사용한다

- 블록요소는 너비(width)와 높이(heihgt)가 지정이 가능하다.

- 블록요소의 너비는 기본적으로 부모의 너비 내에서 최대 너비를 가진다. (기본값 auto)

- 블록요소의 높이는 기본적으로 자신이 가진 컨텐츠 크기 만큼 늘어난다. (기본값 auto)

- 블록 요소는 너비와 상관없이 줄바꿈이 일어난다.

- 블록 요소는 다음에 누가 오는지 상관없이 줄바꿈이 일어난다.

Block(div)

2. 인라인 레벨 요소(Inline Level Element)

- 인라인 레벨 요소는 특정 컨텐츠를 지정할 때 사용한다.

- 인라인 요소는 너비(width)와 높이(heihgt)가 지정이 불가능하다.

- 인라인 요소의 자신이 가진 컨텐츠 크기 만큼 너비와 높이를 가진다. (기본값 auto)

- 인라인 요소는 줄바꿈이 일어나지 않는다.
