CSS position

1. position

요소의 배치 방법을 지정한다.

1-1. 속성값

  • static(기본 값) : 문서의 흐름(기본: 위에서 아래)에 맟춰 나열된다. 위치값 기준이 지정이 안된다.
  • relative : 기본적으로 static처럼 문서의 흐름에 맞춰 나열된다.
    위치값 기준이 문서 흐름상 자신의 원래 위치가 된다.
  • absolute : 위치값 기준이 자신의 상위요소 기준으로 움직인다.
    상위 요소들 position 값이 static 아닌 요소가 기준이 된다(relative, absolute, fixed).
    inline-block과 비슷하게 움직인다.
    float처럼 다른 요소의 위나 아래에 위치한다.
    더 늦게 작성된 요소가 위로 위로 올라온다.
  • fixed : 위치값의 기준이 보이는 화면(뷰 포트) 영역이 기준이다.

2. top, bootom, left, right => offset

position값에 따른 기준이 되는 요소로부터 상하좌우 얼마나 떨어져있는지 지정한다.

3. z-index

요소들이 겹칠때 더 위에 보여줄 요소를 지정할수 있다.
z-index 값이 높으면 위로 올라온다.
