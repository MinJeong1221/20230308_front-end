CSS background

1. background-color

배경색을 지정한다.
속성에 키워드 rgb rgba 16진수 등 넣는다.

2. background-clip

배경이 지정되는 범위를 설정한다.
적용되는 범위이다.(시작되는 범위가 아님)

    속성값

  • border-box(기본값) : border 영역까지 지정
  • padding-box : padding 영역까지 지정
  • content-box : content 영역까지 지정

3. background-image

이미지를 배경으로 지정한다. url("이미지 파일 경로") 형식으로 사용한다.

4. background-repeat

배경 이미지의 반복 방법을 지정한다.

    속성값

  • repeat(기본값) : x축, y축 방향으로 둘다 반복한다.
  • no-repeat : 반복을 하지 않는다.
  • repeat-x : x축으로 방향으로 반복한다.
  • repeat-y : y축으로 방향으로 반복한다.

5. background-position

배경 이미지의 위치를 지정한다.

    속성값

  • x축, y축(숫자로 값을 입력해주면 된다. 음수도 사용가능하다.) 방향으로 지정한다.
  • 한가지 값만 사용시 나머지는 자동으로 center로 지정된다.
  • 키워드(left,right,top,bootom), % 사용가능

6. background-size

배경이미지의 사이즈를 지정한다.

    속성값

  • 가로(x), 세로(y) 값 지정가능하다
  • 값을 하나만 지정하면 가로값이 지정되서 세로는 자동으로 이미지 비율에 맞춰서 지정된다.
  • auto : 원본 이미지 크기만큼 지정한다.
  • contain : 가로 세로 중 더 긴 쪽을 기준으로 사이즈를 맟춘다(이미지가 잘리지 않은 범위에서).
  • cover : 가로 세로 중 더 짧은 쪽을 기준으로 사이즈를 맞춘다(이미자를 자르면서 꽉 채운다).

7. background-origin

배경 이미지가 시작되는 범위를 지정한다.

    속성값

  • content-box(기본값) : content 영역부터 지정
  • border-box : border 영역부터 지정
  • padding-box : padding 영역부터 지정

7. background-attachment

배경 이미지가 시작되는 범위를 지정한다.

    속성값

  • scroll(기본값) : 화면을 스크롤하면 배경이미지도 고정이 되어 스크롤 됩니다.
  • local : 스크롤을 하면 배경이미지가 함께 움직인다.
  • fixed : 뷰포트기준으로 위치가 고정이 된다. 배경이미지로 지정된 요소를 통해서만 보인다.

8. background

배경관련 속성을 한번에 지정하는 단축속성이다.
position과 size는 /로 구분하여 작성(position이 먼저 작성이 된다.)
단축속성으로 size를 작성하고 싶으면 꼭 position값을 지정해야 된다.
