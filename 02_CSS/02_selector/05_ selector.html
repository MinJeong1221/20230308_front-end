가상 클래스 선택자(" :가상 클래스 이름 ")

사용자가 특정 행동을 했을때 가상 클래스가 붙는 것처럼 동작한다.

  • :hover => 마우스가 올라가면 바뀐다.
  • :active => 마우스가 클릭하고 있는 동안 바뀐다.
  • :focus => 클릭이 가능한 요소들이 focus가 되어 있는 동안 바뀐다.
  • :first-child => 첫번째 자식
  • :nth-child(n) => n번째 자식
  • nth-of-type(n) => 앞에 적은 선택자를 가진요소의 첫번째
  • n번째 자식
  • :last-child => 마지막 자식
