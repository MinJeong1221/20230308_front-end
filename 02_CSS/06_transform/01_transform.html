CSS transform

요소를 병형시킬수 있다.
transform의 값은 변형함수를 사용한다.

1. translate(이동)

요소를 현재위치 기준으로 움직인다.

  • transform: translateX(30px); : x축으로 움직인다.
  • transform: translateY(30px); : y축으로 움직인다.
  • transform: translate(30px, 20px); : x, y축으로 움직인다.
  • %를 사용하면 자기자신이 기준이 되면서 움죽인다

2. scale(크기)

요소를 크기를 변형(늘린다)한다.

요소안에 contant도 같이 변형(늘린다)한다.

자기 자신만 변하고 다른요소는 신경쓰지 않는다.

  • transform: translateX(배율 or %); : x축으로 늘린다.
  • transform: translateY(배율 or %); : y축으로 늘린다.
  • transform: translate(배율 or % , 배율 or %); : x, y축으로 늘린다.(값을 하나만 넣으면 x, y둘다 늘어난다.)
  • 단위는 배율이나 %를 사용한다
x
y
xy

3. rotate(회전)

요소를 회전시킨다.
deg(각도) 단위를 사용한다.

  • transform: rotateX(deg); : 지정한 각도만큼 x축으로 돌린다.
  • transform: rotateY(deg); : 지정한 각도만큼 y축으로 돌린다.
  • transform: rotateZ(deg); : 지정한 각도만큼 z축으로 돌린다.
  • transform: rotate(x, y, deg); : 지정한 각도만큼 x축, y축으로 돌린다.
  • transform: rotate3d(x, y, z, deg); : 지정한 각도만큼 x축, y축, z축으로 돌린다.
  • 단위는 deg(각도)를 사용한다.
