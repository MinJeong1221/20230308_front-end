CSS flex

컨테이너 요소가 아이템 요소들을

Flex => container에 사용하는 속성

  • display : flex 컨테이너를 지정합니다.

      속성값

    • flex : flex로 지정(block레벨 요소를 배치)
    • inline-flex : flex로 지정(inline레벨 요소를 배치)
  • flex-direction : 컨테이너 주축 및 방향을 설정

      속성값

    • row(기본값) : 추축을 가로로 지정하고 왼쪽에서 오른쪽으로 배치
    • column : 추축을 세로로 지정하고 위에서 아래로 배치
    • rew-revers : 추축을 가로로 지정하고 오른쪽에서 왼쪽으로 배치
    • column-revers : 추축을 세로로 지정하고 아래에서 위로 배치
  • flex-wrap : 컨테이너 너비보다 항목이 많을경우 줄바꿈의 여부 지정

      속성값

    • nowrap : item을 한줄에 표시
    • wrap : 여러줄에 표시
    • wrap-revers : 여러줄에 표시하되 시작점과 끝점이 바뀜
  • flex-flow : 배치 방향과 줄바꿈을 한번에 지정 (flex-direction, flex-wrap)
  • justify-content :주축에서 item의 정렬방법을 지정

      속성값

  • align-items :교차축에서 item 정렬방법을 지정

      속성값

  • gap : item간의 공간
