DOM(Document Object Model)

html의 각 요소를 다루기 위해서 브라우저가 제공하는 객체

document 객체를 통해서 요소를 가져올 수 있다

getElementById("id값") : 특정 아이디 값을 가지는 요소 1개를 반환한다.

getElementsByClassName("class값") : 특정 클래스를 가지는 요소를 여러개를 반환한다.
=> html collection이라는 유사배열 형태로 반환한다.

querySelector("CSS 선택자") : 선택자에 해당하는 문서상 첫번째 요소를 반환한다.

querySelectorAll("CSS 선택자") : 선택자에 해당하는 모든 요소를 반환한다.
=> NodeList라는 유사배열 형태로 반환한다.
